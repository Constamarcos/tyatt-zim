## Formato de cada artigo

Cada artigo gerado segue o formato do assistente:

```json
{ "id": "malaria", "titulo": "Malária", "categoria": "Saúde", "corpo": "..." }
```

- **Extensão:** 300–600 palavras (artigos "bem completos": o que é, como funciona,
  exemplos práticos, erros comuns, quando procurar ajuda).
- **Linguagem:** português claro, exemplos angolanos (kwanzas, província, época
  das chuvas), sem jargão desnecessário.
- **Primeiro parágrafo autossuficiente** — é o que o assistente lê em voz alta
  como resposta direta.

## Prioridade de geração (por lotes)

1. Saúde (o mais procurado e o mais valioso)
2. Matemática + Português (época de exames)
3. Agricultura + Ofícios (renda das famílias)
4. Restantes categorias
